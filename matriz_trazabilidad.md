# Matriz general de trazabilidad

| Dominio | Referencia principal | Control evaluado |
|---|---|---|
| Autenticacion y credenciales | NIST SSDF / ISO 27001 / OWASP | Proteccion de credenciales, hash de contrasenas y gestion segura de tokens o sesiones. |
| Validacion de entradas | NIST SSDF / OWASP Top 10 / OWASP ASVS | Validacion de entradas, validacion en backend y prevencion de inyeccion. |
| Control de acceso | ISO 27001 / OWASP ASVS / NIST SSDF | Roles, permisos, proteccion de rutas y autorizacion en servidor. |
| Manejo de errores y registros | ISO 27001 / OWASP / NIST SSDF | Manejo seguro de errores, registro de eventos y proteccion de datos sensibles en logs. |
| Calidad de codigo | NIST SSDF / Buenas practicas de calidad | Nombres claros, reduccion de duplicacion y revision de codigo. |
| Componentes y dependencias | NIST SSDF / OWASP Top 10 / ISO 27001 | Librerias actualizadas, dependencias necesarias y riesgo de componentes vulnerables. |
